## International Week - Case 3: Sara

### 1. Prerequisite

*   **Install Node.js:** [https://nodejs.org/](https://nodejs.org/) (Select the **LTS** version).

---

### 2. How to start the prototype

1.  **Download and Extract**
    *   Click the green **<> Code** button and select **Download ZIP**.
    *   Extract the downloaded ZIP file into a folder.

2.  **Open the Terminal**
    *   **Windows:** Right-click the project folder > Open in terminal (if you're not on windows 11, google how to open the terminal)
    *   **Mac:** Right-click the project folder > **Services** > **New Terminal at Folder**.

3.  **Install**
    *   Type `npm install` and press **Enter**. Wait for it to finish.

4.  **Run**
    *   Type `npm start dev` and press **Enter**.

5.  **View**
    *   Open your browser and go to: **localhost:8080**

---

### 3. How to stop / Troubleshooting

*   **To Stop:** Press **CTRL + C** in the terminal window.
